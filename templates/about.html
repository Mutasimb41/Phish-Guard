About Us

PhishGuard is a Final-Year Project built to demonstrate real-time phishing detection for both email (.eml) files and URLs. The backend is built with Flask, the frontend uses Bootstrap 5 with a touch of JavaScript animation, and the core engine is powered by a hybrid deep learning model.
